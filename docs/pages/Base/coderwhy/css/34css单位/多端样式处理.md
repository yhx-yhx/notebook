### rem + 媒体查询 + font-size
> rem 参照 是 html 根元素的大小

```css
@media screen and (min-width:375px ){
    font-size:16px
}
@media screen and (min-width: 750px){
    font-size:18px
}
@media screen and (min-width:1080px){
    font-size:19px
}
```

- 缺陷
  - 需要写大量的媒体查询
  - 不够动态的实时的 变化css
